# ⚡ QUÉ HACER AHORA MISMO

## 🎯 En 3 pasos:

### ✅ PASO 1: Ejecuta los servidores

**Opción A (Recomendada):**
- Haz **doble clic** en: `START_ALL.bat`
- Se abrirán 2 ventanas automáticamente
- Espera 10 segundos

**Opción B (Manual):**
- Abre 2 terminales
- Terminal 1: `npm run dev`
- Terminal 2: `npm run server`

---

### ✅ PASO 2: Abre el navegador

```
http://localhost:5173
```

**Deberías ver:** Página de login con formulario

---

### ✅ PASO 3: Prueba el login

**Ingresa:**
- Usuario: `MARIO`
- Contraseña: `123`

**Haz clic:** "Ingresar al sistema"
